# RJ Link Converter

## 機能

- ページ内の作品番号（RJから始まるもの／「RJ + 6桁以上の数字」パターン）を自動検出
- 検出された作品番号（RJから始まるもの）をDLsiteの該当ページへのリンクに変換
- 既存のリンクやscriptタグ内のテキストは変換しない
- 変換したリンクをクリックすると、その履歴（作品タイトル・RJ番号・クリック日時・クリック元ページ）を記録
- ツールバーの拡張機能アイコンをクリックすると、クリック履歴の一覧を表示・削除できる

## インストール方法

本拡張機能はChromeウェブストアには公開していません。以下の手順でソースコードから直接インストールしてください。Chrome / Edge / Brave / Opera / Vivaldi などの Chromium 系ブラウザに対応しています。

1. このリポジトリを入手します。次のいずれかの方法で取得してください。

   **方法A: ZIPでダウンロード（コマンド操作不要）**

   このページ上部の緑色の「**Code**」ボタン →「**Download ZIP**」をクリックし、ダウンロードしたZIPファイルを任意の場所に解凍します。

   **方法B: `git` コマンドでクローン**

   ```bash
   git clone https://github.com/tebukurokun/rj-link-converter.git
   ```

   **方法C: GitHub CLI（`gh`）でクローン**

   ```bash
   gh repo clone tebukurokun/rj-link-converter
   ```

   > 方法B・Cはコマンド（ターミナル）を使う方法です。あらかじめツールのインストールが必要な場合は、以下の公式ページを参照してください。コマンドに慣れていない場合は方法Aがおすすめです。
   >
   > - Git のインストール: <https://git-scm.com/downloads>
   > - GitHub CLI（`gh`）のインストール: <https://cli.github.com/>（`gh` の利用には Git も必要です）

2. ブラウザのアドレスバーに拡張機能の管理ページのURLを入力して開きます。使用しているブラウザに応じて以下を入力してください。

   | ブラウザ | URL |
   | --- | --- |
   | Chrome | `chrome://extensions` |
   | Edge | `edge://extensions` |
   | Brave | `brave://extensions` |
   | Opera | `opera://extensions` |
   | Vivaldi | `vivaldi://extensions` |

3. 画面右上の「**デベロッパー モード**（開発者モード）」をオンにします。

4. 「**パッケージ化されていない拡張機能を読み込む**」ボタンをクリックします。

5. ダウンロード（またはクローン）した `rj-link-converter` フォルダ（`manifest.json` があるフォルダ）を選択します。

6. 拡張機能一覧に「DLsite RJ Link Converter」が表示されればインストール完了です。任意のページを開くとRJ番号が自動的にリンクに変換されます。

### アップデート方法

最新のソースコードを取得したあと、拡張機能の管理ページで本拡張機能の「再読み込み」ボタン（🔄）をクリックすると最新の状態が反映されます。ソースコードの取得方法は、インストール時にどの方法を使ったかによって異なります。

- **`git` / `gh` でクローンした場合（方法B・C）**: フォルダ内で `git pull` を実行するだけで最新版に更新できます。更新のたびにダウンロードし直す必要がないため、繰り返し使うなら Git を入れておく方法がおすすめです。

  ```bash
  git pull
  ```

- **ZIPでダウンロードした場合（方法A）**: `git pull` は使えません。改めて最新のZIPをダウンロードして解凍し、古いフォルダと置き換えてください。その後、管理ページで「再読み込み」するか、フォルダの場所が変わった場合は一度削除してから読み込み直してください。

なお、バージョンによっては新しい権限（クリック履歴機能で使用する保存領域や DLsite へのアクセスなど）が追加されることがあります。その場合はブラウザが権限の再許可を求めることがあるため、表示された内容を確認して許可してください。

## プライバシー・安全性について

- **ページの読み取り**: 表示中のページの文字を読み取り、RJ番号を探してDLsiteへのリンクに変換します。読み取った内容をどこかに保存・送信することはありません。
- **外部への通信**: 変換したリンクをクリックしたときだけ、作品タイトルを取得するために **RJ番号（例: RJ236867）だけ** をDLsiteに送信します。それ以外の情報や、DLsite以外のサーバーへの送信は一切行いません。
- **保存されるもの**: クリック履歴（作品タイトル・RJ番号・日時・クリック元ページ）は **お使いのブラウザ内だけ** に保存されます。外部に送られることはなく、作成者を含め他人が見ることはできません。
- **アカウント・追跡なし**: ログインや会員登録は不要です。広告の表示や、利用状況を収集するような追跡（トラッキング）は行いません。
- **透明性**: ソースコードはすべて公開されており、誰でも中身を確認できます。

### クリック履歴の確認・削除方法

- ブラウザ右上のツールバーにある本拡張機能のアイコンをクリックすると、クリック履歴の一覧が表示されます。
- 履歴を消したいときは、その画面の「**履歴を削除**」ボタンを押してください。すべての履歴がブラウザ内から削除されます。
